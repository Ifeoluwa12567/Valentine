# Valentine Card

A simple, single-page Valentine card built with HTML, CSS, and JavaScript.

## Overview
This project displays a playful Valentine message with interactive buttons.
It is designed to be lightweight, mobile-friendly, and easy to share as a link.

## Features
- Single HTML file
- Responsive card layout
- Simple animations and interactions
- No external libraries or frameworks

## How to Use
1. Open the page in a web browser, or
2. Deploy it using a static hosting service (e.g. GitHub Pages, Vercel, Netlify).

## Customization
You can edit:
- The displayed name
- The text messages
- Colors and styling in the CSS section

All changes can be made directly in the HTML file.
